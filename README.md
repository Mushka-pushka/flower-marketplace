# Онлайн-маркетплейс цветочной продукции

**Разработка онлайн-маркетплейса цветочной продукции с семантическим поиском и асинхронной обработкой заказов**

## Технологии

- **Backend:** Go
- **Frontend:** React + TypeScript
- **База данных:** PostgreSQL 16
- **Кэширование:** Valkey
- **Брокер сообщений:** RabbitMQ
- **Контейнеризация:** Docker + Docker Compose

## Функциональные возможности

- Семантический поиск по тегам (эмоции, ситуации)
- Асинхронная обработка заказов через RabbitMQ
- Роли: покупатель, продавец, администратор
- Личные кабинеты с историей заказов
- Отзывы и избранное
